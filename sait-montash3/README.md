## Как добавить фото в галерею

### Частные лица
1. Положите фото в папку `photo/chasniki/`
2. Откройте файл `photo/photos.js`
3. Добавьте имя файла в массив `chasnikiPhotos`:

```javascript
const chasnikiPhotos = [
    'existing_photo.jpg',
    'new_photo.jpg',  // Добавьте сюда
    'another.jpg'
];
```

### Юр лица
1. Положите фото в папку `photo/yriki/`
2. Добавьте имя файла в массив `yrikiPhotos` в файле `photo/photos.js`

## Настройка EmailJS

### 1. Регистрация
Зарегистрируйтесь на [emailjs.com](https://www.emailjs.com)

### 2. Подключение Email сервиса
1. Войдите в дашборд EmailJS
2. **Email Services** → **Add Email Service**
3. Выберите Gmail и подключите аккаунт

### 3. Создание шаблона
1. **Email Templates** → **Create New Template**
2. Используйте шаблон из файла `email-template.html`
3. Сохраните и скопируйте **Template ID**

### 4. Настройка в коде
В файле `script.js` найдите и замените данные:

```javascript
// Строка инициализации
emailjs.init("ВАШ_PUBLIC_KEY");

// Отправка формы
emailjs.sendForm('ВАШ_SERVICE_ID', 'ВАШ_TEMPLATE_ID', callbackForm)
```
