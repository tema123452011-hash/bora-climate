# BORA Climate — Проект сайта

## 📋 Описание

Одностраничный сайт компании по монтажу и обслуживанию климатических систем (кондиционеры, вентиляция) в Санкт-Петербурге.

## 🎯 Основные особенности

Сайт обладает следующими функциональными возможностями:

- **Табы услуг** — переключение между Кондиционированием, Сервисом и Вентиляцией без перезагрузки страницы
- **Галерея работ** — динамическая загрузка фотографий из JavaScript-массивов с разделением на частных и юридических лиц
- **Модальное окно** — форма обратной связи с полями: имя, телефон, email, сообщение
- **Отправка писем** — используется библиотека EmailJS для отправки данных на почту без сервера
- **Адаптивность** — три точки разрыва: мобильные (< 768px), планшеты (< 992px), десктоп (> 992px)
- **Защита от спама** — реализована через honeypot-поле и чекбокс "Я не робот"
- **Плавная прокрутка** — при клике на услуги страница плавно прокручивается к нужному блоку

### Используемые технологии

- **HTML5** — семантическая разметка
- **CSS3** — стили, CSS-переменные, медиа-запросы для адаптива
- **JavaScript (ES6+)** — логика переключения табов, модальное окно, отправка формы
- **EmailJS** — отправка писем через API
- **Google Fonts (Roboto)** — шрифт для текста

## 📖 Пояснения к коду

### 1. Табы услуг (index.html + script.js)
Логика работы: при клике на кнопку получаем ID из атрибута `data-target`, находим блок с таким ID и показываем его. Если нажать на активную кнопку ещё раз — блок скрывается. Используется `scrollIntoView` для плавной прокрутки.

**Почему так сделано:** Простое решение без библиотек, работает быстро, легко поддерживать.

### 2. Галерея (script.js + photo/photos.js)
Фотографии хранятся в массивах JavaScript в отдельном файле. При загрузке страницы скрипт генерирует HTML-теги `<img>` для каждого фото.

**Почему так сделано:** Не нужно редактировать HTML при добавлении фото — достаточно добавить имя файла в массив.

**Важно:** Если фото не загрузится (файл не найден), сработает `onerror` и блок скроется.

### 3. Модальное окно и форма (index.html + script.js)
Форма отправляется через EmailJS. Данные уходят на почту без использования сервера.

**Защита от спама:**
- **Honeypot** — скрытое поле `website`. Если его заполнить — отправка блокируется (боты часто заполняют все поля).
- **Чекбокс "Я не робот"** — простая проверка, которую люди легко ставят, а сложные боты могут пропустить.

### 4. CSS-переменные (styles.css)
Цвета вынесены в `:root` — можно легко поменять цветовую схему всего сайта, изменив значения в одном месте.

```css
:root {
    --primary: #0066cc;  /* Основной цвет */
    /* ... */
}
```

## 🔒 Безопасность

На сайте реализованы следующие меры безопасности:

| Мера | Описание |
|------|----------|
| **Honeypot** | Скрытое поле для отлова ботов |
| **Валидация формы** | Поля `required` и проверка типов (email, tel) |
| **Отключение автозаполнения honeypot** | `autocomplete="off"` |
| **Блокировка повторной отправки** | Кнопка блокируется до завершения отправки |

### ⚠️ На что обратить внимание

1. **EmailJS ключ** — публичный ключ `lQnwTxuPAi8YXEnxK` безопасен для использования на фронтенде. Но если станет известен, злоумышленник может отправить письма от вашего имени. Можно ограничить домены в настройках EmailJS.

2. **Телефон и email** — данные контактов видны в исходном коде. Это нормально для сайтов-визиток, но не подходит для sensitive данных.

3. **Нет серверной валидации** — данные из формы уходят напрямую. В будущем при подключении сервера (PHP/Node.js) рекомендуется добавить серверную проверку.

## 🚀 Как запустить проект

1. **Простой запуск:** Просто откройте файл `index.html` в любом браузере
2. **Рекомендуемый способ:** Используйте локальный сервер (например, расширение Live Server в VS Code)

### Как проверить, что всё работает:

1. Откройте страницу в браузере
2. Кликните на кнопки-табы (Кондиционирование, Сервис, Вентиляция) — должен открываться нужный блок
3. Кликните на "Наша работа" — появится галерея
4. Нажмите "Обратная связь" — откроется форма
5. Заполните форму и нажмите "Отправить" — появится сообщение об успехе
